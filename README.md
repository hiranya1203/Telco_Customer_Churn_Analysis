# 📊 Telco Customer Churn Analysis

## 🔍 Project Overview
End-to-end customer churn analysis
identifying why customers leave and
defining data-driven actions to
retain them.

**Overall Churn Rate Found: 26.54%**
**Tools Used: Python | MySQL | Power BI**

---

## 💡 Key Findings
1. **26.54% overall churn rate** — 1 in 4 customers is leaving
2. **Month-to-month customers churn at 42%** — 14x higher than two-year contract holders
3. **New customers (0-12m) churn at 47%** — broken onboarding is the #1 retention risk
4. **Fiber optic customers churn at 42%** despite paying premium prices — price vs quality gap
5. **Customers with no TechSupport and no OnlineSecurity churn at 48.96%** vs 9.01% for feature users

---

## 📋 Churn Intervention Playbook

| Trigger | Action | Priority |
|---|---|---|
| tenure < 12m + Month-to-month | Send retention offer in 24hrs | 🔴 Urgent |
| High value + Churned | Assign dedicated account manager | 🟠 High |
| No TechSupport + No Security | Offer free 30-day feature trial | 🟡 Medium |
| Fiber optic + tenure < 24m | Proactive quality check call | 🟡 Medium |
